  • MyFotos

    Foto Verwaltung für OSX, Windows und Linux. Liest Fotos aus Verzeichnissen, erstellt Arbeitskopien, lässt diese bearbeiten und erstellt Alben.

    Pure Webtechnologien mit neuen CSS3 Animationen mit Node-Webkit als native Applikation für Windows, OSX und Linux.

  • MyFeed

    Server Software ähnlich Wordpress zum einfachen publizieren von Markdown-Texten. Parsed Links und erstellt Media Attachments. Kann RSS & MyFeeds lesen und als Timeline darstellen.

    IO.js/Node.js Applipkation mit integrierter File-PouchDB. Parsing von Webseiten per Links nach Social Graph Inhalten.

    MyFeed von Robert Fork on Github

  • Rolling Dice

    OSX download

  • Todo

    Erstelle kostenlos teilbare Echtzeit Todo-Listen. Teile den Link mit deinen Freunden und arbeite Gemeinsam. Kein Login, keine Kosten. Kann, gefüttert mit Links zu Medien, als Playlist für "MyPlayer" fungieren.

    AngularJS für das Frontend, Socket.io für Kommunikation mit dem nodejs Server. Icons von Marie Schweiz

  • deZem Reports

    Tool mit dem die Nutzer von deZem ihre eigenen Reporte erstellen können. Diese werden dann in Intervallen per E-Mail an die Nutzer versendet.

    deZem besuchen

  • Cloud Tool

    Single Page Web-App im Auftrag der Firma Pinuts für Biotronik, die es einem Benutzer erlaubt Dateien bzw. Medien hochzuladen, zu verwalten und zu konsumieren. Änlich wie man das von Google Docs her kennt.

    Die App besteht im Frontend größenteils aus Vanilla Javascript mit jQuery Unterstützung für die Selektor Engine. Für die Templates hab ich John Resigs Mini-Template Snippet verwendet.

  • Shop-Backend

    Ich habe zum Programmieren auf CoffeeScript, Less und Bootstrap gesetzt. GruntJS zum Kompilieren der Source Files.

  • Kolekt

    Kolekt ist eine Art Twitter Clone mit dem Fokus auf Nachrichten und der Konversation. Idee dahinter ist, dass jede Nachricht den Beginn einer Idee/Konversation darstellen kann. Abgesetzte Nachrichten sind so optimalerweise immer Teil einer anderen Konversation.

    Die iOS App ist in nativem Obj-C geschrieben und stützt sich im Backend auf den Parse-Service.
